# 未名大赋

20260712，19：13，诗本无名，

20260714，23：31，因而未名： // 陆续，把我的诗歌携带上时间戳都公开出来


```
若夫玄青先设一层无名的次序，银素再把次序分作宽窄不同的数面，石绿与栀黄各执半部而互不相属；
铅白专司迟缓的连缀，砺黑专司突然的转折，藕紫与铜青则时在句首，时退句腰，时又隐入长长的尾音。

其初，咿弥与罗呢连续相承，柔声沿着绵密的韵脚一重一重展开；
其继，刻塔与柘帛从内部相抵，硬声在柔声腹内留下数处不齐的空格。

空格既成，前部并未真正中止，后部却已先行；后部稍稍增密，前部又从背面折回，于是首尾开始彼此误认。
// 青黛缓缓回转，银灰重新相合，栀白在二者之间停留片刻，而后仍向未名之处退去。
```


```
尔乃米黄逐渐变薄，绀蓝逐渐变厚，诸色虽然不再依照旧有的次序排列，却仍保留旧次序隐约的轮廓；
软铅向内部缓缓收拢，素绢向外部逐层展开，收与展彼此错过，又在更远的一层重新相遇。

一度只有罗呢与咿弥彼此牵连，长音沿着长音扩展，句身因此宽阔而几乎没有突出的角；
一度只有砺柘与刻塔彼此抵触，短促之音接连进入，整篇因而分作数段始终不能完成的篇章。

数段各自向前延伸，却都在同一尾部忽然折回；数段各自暂时停住，却又都从同一首部重新生长；
// 青黛缓缓回转，银灰重新相合，栀白在二者之间停留片刻，而后仍向未名之处退去。
```

```
及其第三次回返，铜青不再安居于边缘，藕紫不再安居于中央，石绿与铅白已经交换了各自半副位置；
绛灰从素白之后略略显出，米黄从玄青之内逐层退去，所有界线都比先前更加柔软，却也更加难以取消。

此时长句不再求取齐整，有的延至二十余字仍然保有余势，有的才过十数字便忽然转入另一条未定的方向；
较短的句段也不骤然终止，总携带一段旧有的回声、一枚曾经出现的颜色、一层尚未完成的侧纹重新进入。

来者并非全然新生，去者也未曾全然消失，相似之处因此可以辨认，细微差异因此可以被反复感受；
// 青黛缓缓回转，银灰重新相合，栀白在二者之间停留稍久，而后才向未名之处退去。
```
```
既而四部同时来至：弥蓝与绵罗居于前部，砺白与塔黑居于后部，铜青与藕紫分处两个不断移动的侧面；
四部并不争夺主次，只把疏密互相交换，把长短互相迁移，使一片无名之中逐渐生出深处、近处与不可跨越的里层。

深处并不等于内部，近处也不等于眼前；不过某些声音被反复听见，某些颜色被反复认出，某些未闭的关系被反复等待；
于是在没有对象之处出现了层次，在没有事件之处出现了趋势，在没有叙述之处出现了回返，在没有名字之处出现了期待。

末后诸色逐渐删简，诸声逐渐疏少，先退砺黑，再退绛灰，继而石绿与铜青也各自还回较远的位置；
只余咿弥、罗呢、银素、青黛，彼此不能解释彼此，却共同维持最后一段迟迟不肯落定的长势。

// 青黛仍旧回转，银灰仍旧相合，栀白此次不再停留；一切似已收束，却又从尾音的背面重新开始。
```
