# Architecture

This directory stores architecture design documents and decision records for the ShepherdQR GitHub Pages site.

## Documents

- [Site Content Architecture](site-content-architecture.md): Design decision for unifying Markdown, HTML rendering, homepage structure, metadata, and migration strategy.
- [Markdown Publishing Workflow](markdown-publishing-workflow.md): Current Markdown-first authoring, build, and article rendering workflow.
- [Legacy HTML to Markdown Migration](legacy-html-to-markdown-migration.md): Migration design for converting historical HTML notes into Markdown while preserving original date information.
- [Legacy Migration Pilot Report](migration-pilot-report.md): First audit and preview report for the legacy HTML-to-Markdown migration tool.
- [Full Markdown Migration Report](full-markdown-migration-report.md): Bulk conversion result after moving public article data to Markdown sources.
- [Archive and Article Page Design](archive-and-article-page-design.md): Shared article reader and archive/category page design.
- [Legacy HTML Cleanup Strategy](legacy-html-cleanup-strategy.md): Accepted strategy for deleting old HTML compatibility copies after Markdown verification.
- [Project Change and Migration Report](project-change-migration-report.md): Ongoing project-level ledger for architecture changes, migrations, cleanup decisions, and verification.
