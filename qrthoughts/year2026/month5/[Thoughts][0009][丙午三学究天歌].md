# 丙午三学究天歌
- 【20260216】过年回家的火车上，我构思本诗，于晚上完成。
- 【20260502】原18韵，今日重新整理为20韵。

```
丙午启元章，东风遍八荒。
星枢回斗柄，岁律焕瑶光。
平生耽玄理，格物探微方。
全息印真如，纠缠度穹苍。

辛流含动静，拉氏正典章。
序畴穷寂旷，同伦辨微茫。
上同窥不变，函子统界疆。
自指生奇限，涌现辟新航。

型为天地骨，算即古今纲。
数定乾坤极，理穷万象昌。
机缄藏隐序，三学辩濠梁。
欲通造化妙，岂畏路悠长！

驰心归太素，骋思任翱翔。
晨昏研奥义，坐忘对星霜。
环中穷至赜，天籁自宫商。
不羡尘间贵，唯昭真理彰。
新岁增雄砺，长风助骏骧。
平生无限意，尽付道心芒。
```
